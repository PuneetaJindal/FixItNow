# 🛠️ FixItNow – Home Service Booking Platform

## 📌 Overview
FixItNow is a full-stack web application that allows users to book home services such as plumbing, electrical repair, AC servicing, and cleaning. It provides a role-based system where users can book services and operators can manage and accept jobs.

This project is built using Django, Django REST Framework, HTML, CSS, and JavaScript (Fetch API) and is fully deployed on Render.
